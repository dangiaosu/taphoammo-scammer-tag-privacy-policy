Privacy Policy for TapHoaMMO Scammer Tag

Last updated: March 28, 2025

TapHoaMMO Scammer Tag ("the Extension") is a Chrome extension designed to display a "Scammer" tag with a reason next to seller names on taphoammo.net, helping users avoid potential scammers. This Privacy Policy explains how we handle user data.

1. Data We Collect

  • Web History: The Extension accesses the URL of the current tab (e.g., https://taphoammo.net/danh-muc/...) to determine the type of page (category, user info, or product page) and display the "Scammer" tag accordingly.
  • Website Content: The Extension reads seller names on taphoammo.net pages to compare them with a scammer list and display a warning tag if a match is found.
  • We do not collect any personally identifiable information, health information, financial information, authentication information, personal communications, location, or user activity.

2. How We Use Data

The collected data (URL and seller names) is used solely to display the "Scammer" tag and reason on taphoammo.net pages, fulfilling the Extension's purpose of protecting users.

The scammer list is fetched from Google Apps Script APIs and stored locally in Chrome Storage to reduce API calls.

3. Data Sharing

  • We do not sell, transfer, or share any user data with third parties.
  • We do not use data for purposes unrelated to the Extension's functionality.
  • We do not use data to determine creditworthiness or for lending purposes.

4. Data Storage

The scammer list and last updated timestamp are stored locally in Chrome Storage on the user's device. This data is not transmitted to any external servers.

5. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated "Last updated" date.
